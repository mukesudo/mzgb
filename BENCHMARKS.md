## Engine Selection (v0.3)

mzgb v0.3 automatically selects the fastest available engine:

| Pattern type | Engine | Notes |
|---|---|---|
| Single literal (no regex metacharacters) | **Boyer-Moore** (`str.find`) | Python's `str.find` uses a Boyer-Moore-Horspool variant |
| Single regex pattern | **Bloom + regex** | Trigram Bloom gate before `re.search` (requires `pybloom-live`) |
| N ≥ 2 literal patterns | **Aho-Corasick** | O(n) scan via `pyahocorasick` (requires `mzgb[fast]`) |
| N ≥ 2 regex patterns | **Multi-regex** | Single compiled alternation `(?:p1)\|(?:p2)\|…` |
| No pattern | **Always** | No matching overhead |

Install optional engines:

```bash
pip install "mzgb[fast]"   # Aho-Corasick + Bloom
pip install "mzgb[drain]"  # Drain3 template clustering
pip install "mzgb[all]"    # Everything
```
