Usage

One command.
Infinite power.

Works with files, pipes, kubectl, journalctl — anything that produces text.

Filter by level
$ logsnap --level ERROR app.log
[ERROR] 14:00:04 Connection timeout
[ERROR] 14:00:06 DB query failed
[ERROR] 14:00:09 File not found
Pattern match + context
$ logsnap --pattern "timeout" -C 2 app.log
[INFO] 14:00:03 Retrying connection...
[ERROR] 14:00:04 Connection timeout
[WARN] 14:00:05 Backoff: 5s
Time range filter
$ logsnap --from "14:00" --to "14:30" app.log
[INFO] 14:00:01 Server started
[INFO] 14:15:22 User login
[WARN] 14:28:05 High memory: 85%
Pipe from anywhere
$ kubectl logs pod/api | logsnap --level ERROR
$ cat *.log | logsnap --pattern "OOM"
$ journalctl | logsnap --summary

Get Started

Up and running
in 30 seconds

Clone and run. No install required.

install
# Clone the repo
$ git clone https://github.com/mukesudo/logsnap.git
$ cd logsnap

# Run immediately (no install needed)
$ python3 -m logsnap --level ERROR app.log

# Or run the demo
$ ./demo.sh
