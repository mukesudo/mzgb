mzgb

A CLI tool for filtering large log files by level, pattern, and time range. Streams line by line — no memory issues with large files.

Get started View source
terminal
$

Usage

Filter by log level
$ mzgb --level ERROR app.log [ERROR] 14:00:04 Connection timeout [ERROR] 14:00:06 DB query failed [ERROR] 14:00:09 File not found
Pattern search with context
$ mzgb --pattern "timeout" -C 1 app.log [INFO] 14:00:03 Retrying connection... [ERROR] 14:00:04 Connection timeout [WARN] 14:00:05 Backoff: 5s
Filter by time range
$ mzgb --from "14:00" --to "14:30" app.log [INFO] 14:00:01 Server started [INFO] 14:15:22 User login [WARN] 14:28:05 High memory: 85%
Works with pipes
$ kubectl logs pod/api | mzgb --level ERROR $ journalctl | mzgb --summary $ cat *.log | mzgb --pattern "OOM"

Options

--levelFilter by log level (ERROR, WARN, INFO). Repeatable.
--patternFilter by keyword or regex.
--from / --toTime range filter.
-C / --contextShow N lines before and after each match.
--summaryPrint a summary table instead of raw lines.
--followFollow file for new lines (like tail -f).

Install

From source
$ git clone https://github.com/mukesudo/mzgb.git $ cd mzgb $ python3 -m mzgb --help
pipx — isolated install (recommended)
$ pipx install mzgb
pip
$ pip install mzgb
Homebrew — macOS
$ brew tap mukesudo/mzgb $ brew install mzgb
Scoop — Windows
$ scoop bucket add mzgb https://github.com/mukesudo/scoop-mzgb $ scoop install mzgb
Snap — Linux
$ snap install mzgb --classic
Nix
$ nix run github:mukesudo/mzgb
